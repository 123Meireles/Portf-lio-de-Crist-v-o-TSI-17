Minhas Habilidades

HTML

HTML (HyperText Markup Language) é a linguagem de marcação usada para estruturar conteúdos na web.

Desenvolvimento de Software

Ser um "IT" (ou "IT person") refere-se a ser um profissional da área de Tecnologia da Informação.

Layout e Design Responsivo

CSS (Cascading Style Sheets) é uma linguagem usada para estilizar e formatar a apresentação de páginas web. Ela define aspectos como layout, cores, fontes e espaçamentos.

Administração de redes

Administração de redes é o conjunto de atividades, práticas e processos responsáveis pela configuração, monitoramento, manutenção e otimização de redes de computadores.

Windows

Windows é um sistema operacional desenvolvido pela Microsoft, projetado para gerenciar hardware e software de computadores, além de fornecer uma interface gráfica amigável para os usuários.

Terminal

Terminal é um programa ou interface de linha de comando que permite aos usuários interagir diretamente com o sistema operacional por meio de comandos de texto.
